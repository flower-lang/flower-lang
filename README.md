flower lang
===============

# Install


1. `sudo npm install -g flower-lang`
2. `flower-lang`
3. Open <http://localhost:8888>




# Developer

## Build

nekoをインストール

```
>> neko build.n
```

## Start

```
npm start
```

Please access http://localhost:8888
